## Requirements
1. Python 3
2. PIP (package manager)
3. Chrome/Firefox driver
4. Install requirement.txt file using PIP. 
   
   `pip install -r requirements.txt `
 
2. Run the `streamlit app` file.

   ```
   streamlit run app.py
   ```
